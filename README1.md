# Zingle Backend

Bu repo Zingle uygulamasının backend’ini içerir. Tek dosya ile migration, auth, wallet, gift, live, PK ve admin endpointleri hazırdır.

## Gereksinimler

- Node.js v18+
- PostgreSQL
- Redis
- NPM veya Yarn
- PM2 (Production için önerilir)
